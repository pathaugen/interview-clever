Patrick Haugen's Interview Project for Clever

Notes About This Project

Simple framework to show off competency in various categories:

  • Full-Stack Development
  • Python backend
  • Javascript and multiple APIs (jQuery, etc.)
  • AJAX methods with caching, load inticators, etc.
  • JSON usage and manipulation techniques
  • HTML5 and separation of HTML from other code
  • CSS3 and separation of CSS from other code

Project hosted with full source available at:
https://github.com/pathaugen/interview-clever/

Artificial delay added on AJAX loading of sections to be able to view the working code in action (realtime is near-instant). This artificial delay is per-element on HTML load and a random value between 0 - 3 seconds with a new value selected each load. Experience how fast the cache is by flipping between the two menu items (Home, Clever, and Changelog) after the initial random delay load of both.

Recommend bringing up the Javascript Console (F12 in Internet Explorer) and reloading the framework to view all JS log messages to best view content flow, AJAX loading of various files, and development documentation stylings.
